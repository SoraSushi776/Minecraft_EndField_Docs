# 📦 协议核心的物品运输

> 协议核心配合协议存储箱、传送带等设备，构建完整的自动化物流运输体系。

---

## 演示

![协议核心物品运输演示](./images/protocol_transport.gif)

> 若要从协议核心输出物品，你需要`输出端口`和`协议核心端口`
>
> 如图所示进行摆放，即可实现从协议核心运输物品出来

> 注：默认情况下，协议核心的仓库和整个仓库是分开的，如果你需要从整个仓库中取出物品，请使用`仓库存取线基段`和`协议取货口`
---

## 物流体系组成

| 设备 | 作用 |
|------|------|
| [协议核心](/设备/协议核心) | 物流网络的枢纽，制造各类物流设备 |
| [协议存储箱](/设备/协议存储箱) | 物品的存储节点 |
| [仓库存取线基段](/设备/仓库存取线基段) | 连接存储设备的基础线路 |
| [协议存货口](/设备/协议存货口) | 物品存入接口 |
| [协议取货口](/设备/协议取货口) | 物品取出接口 |
| [传送带](/设备/传送带) | 物品的水平运输 |
| [物流桥](/设备/物流桥) | 跨越障碍的运输通道 |
| [汇流器](/设备/汇流器) | 多路物品合并 |
| [分流器](/设备/分流器) | 物品分路输送 |
| [中继器](/设备/中继器) | 扩展协议信号范围 |
| [供电桩](/设备/供电桩) | 为物流设备供电 |

---

## 典型物流线路

```
采集设备 → 传送带 → 协议存货口 → 协议存储箱
                                      ↓
                              仓库存取线基段
                                      ↓
                  协议取货口 → 传送带 → 生产设备
```
