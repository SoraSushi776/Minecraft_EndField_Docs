# 🥘 农夫乐事 (Farmer's Delight) 玩法指南

> 农夫乐事是一款温和拓展 Minecraft 农业与烹饪玩法的模组。通过简易的烹饪系统和丰富的食材，你可以制作从三明治到炖菜、从甜点到盛宴的各类美食。在"我的世界：终末地工业"中，农夫乐事与终末地种植/采种设备深度联动。

## 一、核心特性

农夫乐事的核心设计理念是"温和扩展"——不改变原版游戏的核心体验，而是增加更多烹饪的可能性。

- **新土壤**：沃土（Rich Soil）—— 加速作物生长
- **新作物**：6种特有作物
- **新设备**：厨锅、砧板、煎锅、篮子
- **新食物**：数十种新菜肴，提供各种效果
- **兼容性强**：可与其他模组食材联动烹饪

---

## 二、土壤与耕作

### 沃土（Rich Soil）

| 特性 | 说明 |
|------|------|
| **合成** | 泥土 + 肥料 → 沃土 |
| **效果** | 上方作物生长速度显著加快 |
| **获取方式** | 用有机材料（堆肥/肥料）改良普通泥土 |
| **参考** | [MC百科](https://www.mcmod.cn/item/381986.html) |

### 乐田（Farmland）

- 在沃土上使用锄头开垦 → 乐田
- 乐田比普通耕地更不容易被踩坏

## 五、主要设备

### 🍳 厨锅（Cooking Pot）

核心烹饪设备，用于制作各类菜肴和饮品。

| 特性 | 说明 |
|------|------|
| **使用方式** | 右击打开 GUI，放入食材和水 |
| **热源** | 下方需放置热源（熔炉/营火/岩浆块） |
| **输出** | 烹饪完成后获得菜肴（用碗/瓶/杯盛装） |
| **参考** | [MC百科](https://www.mcmod.cn/item/382007.html) |

**厨锅配方类型**：
- 🥣 **炖菜/汤类**：需用碗盛装
- 🥤 **饮品**：需用玻璃瓶盛装
- 🍝 **意面/米饭**：需用碗盛装
- 🥗 **沙拉**：需用碗盛装

### 🔪 砧板（Cutting Board）

食材处理设备，用于切分食材。

| 特性 | 说明 |
|------|------|
| **使用方式** | 手持刀右击砧板上的食材 |
| **用途** | 将食材切割成更小的部分 |
| **参考** | [MC百科](https://www.mcmod.cn/item/382008.html) |

**砧板处理示例**：
- 面包 → 面包片
- 牛肉 → 牛肉排
- 火腿 → 火腿片
- 卷心菜 → 卷心菜叶
- 番茄 → 番茄片
- 洋葱 → 洋葱碎

### 🍳 煎锅（Skillet）

| 特性 | 说明 |
|------|------|
| **用途** | 煎烤食物，右击可烹饪食材 |
| **参考** | [MC百科](https://www.mcmod.cn/item/382011.html) |

### 🧺 篮子（Basket）

| 特性 | 说明 |
|------|------|
| **用途** | 储存类方块，可自动收集上方掉落的物品 |
| **参考** | [MC百科](https://www.mcmod.cn/item/382010.html) |
