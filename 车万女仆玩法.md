# 🎀 车万女仆 (Touhou Little Maid) 玩法指南

> 车万女仆是以东方Project为主题的 Minecraft 宠物/女仆模组。你可以收集材料建造祭坛、召唤并驯服女仆，通过饰品和工作系统培养她们，让女仆协助战斗、耕作、管理家园。

## 二、获取女仆

### 收集材料

在夜晚的平原生物群系击败以下生物获得关键材料：
- **妖精女仆** — 掉落 P点 等材料
- **森近霖之助** — 掉落重要合成材料

### 建造祭坛

使用以下材料构建多方块祭坛结构：
```
煤炭块 ×？  铁块 ×？  金块 ×？
青金石块 ×？  红石块 ×？  钻石 ×？
P点 ×？
```

> 祭坛是模组中最核心的多方块结构，用于合成绝大部分模组物品。

### 召唤与驯服

1. 在祭坛上举行仪式召唤女仆
2. 使用 **蛋糕** 右键点击女仆将其驯服
3. 驯服后右键打开女仆 GUI 进行各种设置

## 三、女仆系统

### 交互操作

| 操作 | 功能 |
|------|------|
| **右键** | 打开女仆 GUI 界面 |
| **Shift + 右键** | 快速切换待命/工作模式 |
| **使用蛋糕** | 驯服女仆 |

### 好感度系统

女仆拥有好感度，通过互动和赠礼提升。高好感度女仆工作更积极。

### 模型切换

女仆拥有多种皮肤模型，可在游戏内通过下载模型包进行切换和自定义。

---

## 四、女仆饰品与附件

### 保护类

| 饰品 | 效果 |
|------|------|
| **旗指物** | 提升理智和防御，降低受到的玩家伤害 |
| **替身地藏** | 为女仆提供保护 |
| **绀珠之药** | 保护类饰品 |
| **闪避布** | 提供闪避能力 |

各类饰品可提供对爆炸、火焰、弹射物、药水、摔落、溺水等伤害的保护。

### 功能类

| 饰品 | 效果 |
|------|------|
| **女仆背包** | 增加存储空间 |
| **行李箱** | 大容量存储 |
| **博丽的御币** | 东方主题装饰/功能物品 |
| **魔理沙的扫帚** | 东方主题装饰/功能物品 |
| **猫车/轮椅/黄包车** | 女仆可乘坐的载具 |

---

## 五、工作系统

女仆可通过 GUI 设置工作模式，帮助完成自动化任务：

| 工作类型 | 说明 |
|---------|------|
| **种田** | 自动种植和收获作物 |
| **剪羊毛** | 自动为绵羊剪毛 |
| **喂食动物** | 自动喂养动物 |
| **繁殖动物** | 自动进行动物繁殖 |
| **战斗** | 装备武器后协助战斗 |

---

## 六、装饰与家具

模组添加了大量东方Project主题的装饰方块：

| 方块 | 功能 |
|------|------|
| **神社庭灯** | 消耗 P点为女仆提供信标增益效果 |
| **祭坛** | 多方块核心结构 |
| **多方块雕像** | 大型装饰建筑 |
| **女仆床** | 装饰家具 |
| **电子琴/书桌/直播站** | 装饰家具 |
| **坐垫/鸟居** | 装饰家具 |

---

## 七、抽卡系统

| 项目 | 说明 |
|------|------|
| **扭蛋机** | 抽卡设备 |
| **扭蛋** | 消耗品 |
| **模型券** | 兑换新模型 |
| **扭蛋币** | 抽卡货币 |

服主可自定义抽卡概率，玩家可通过抽卡获得不同的女仆模型。

## 八、自定义扩展
