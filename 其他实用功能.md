## 一、经济系统 — CEconomy

CEconomy 是一款经济管理模组，为服务器引入完整的货币和经济体系。

### 核心功能

| 功能 | 说明 |
|------|------|
| **货币系统** | 每位玩家拥有独立的账户余额 |
| **转账** | 玩家之间可以互相转账 |
| **交易** | 支持玩家间点对点交易 |
| **商店** | 创建和管理商店 |
| **经济排行** | 查看服务器富豪榜 |
| **管理员功能** | 管理员可查询/修改经济参数 |

### 常用指令

| 指令 | 功能 |
|------|------|
| `/balance` 或 `/money` | 查看余额 |
| `/pay <玩家> <金额>` | 转账给其他玩家 |
| `/balancetop` | 查看经济排行 |
| `/trade <玩家>` | 发起交易请求 |

### 经济用途

- 玩家间交易物品
- 购买商店商品
- 支付传送等服务费用
- 服务器活动奖励

## 二、传送系统 — HuskHomes

HuskHomes 提供跨维度的传送功能，方便玩家在广阔的 BOP 世界中快速移动。

### 核心功能

| 功能 | 说明 |
|------|------|
| **设置家** | 在任意位置设置个人传送点 |
| **传送回家** | 随时传送回已设置的家 |
| **传送请求** | 请求传送到其他玩家位置 |
| **传送接受/拒绝** | 控制是否接受传送请求 |
| **随机传送** | 随机传送到世界中的安全位置 |
| **传送点** | 公共传送点（管理员设置） |
| **跨维度传送** | 支持在不同维度之间传送 |

### 常用指令

| 指令 | 功能 |
|------|------|
| `/sethome <名称>` | 设置家（可设置多个） |
| `/home <名称>` | 传送回家 |
| `/homes` | 查看所有家的列表 |
| `/tpa <玩家>` | 请求传送到该玩家 |
| `/tpahere <玩家>` | 请求该玩家传送过来 |
| `/tpaccept` | 接受传送请求 |
| `/tpdeny` | 拒绝传送请求 |
| `/rtp` | 随机传送 |
| `/spawn` | 传送到出生点 |
| `/warp <名称>` | 传送到公共传送点 |
| `/back` | 返回上一个位置 |

## 三、储物保护 — ChestLocker

ChestLocker 是一款基于 UUID 和密码的储物箱保护模组，让你可以安全地锁定自己的箱子。

> **Modrinth**: [ChestLocker](https://modrinth.com/mod/chestlocker)

### 核心功能

| 功能 | 说明 |
|------|------|
| **锁定储物箱** | 制作钥匙并设置密码，锁定任何储物箱 |
| **钥匙绑定** | 每个钥匙绑定创建者的 UUID 和密码哈希 |
| **白名单共享** | 添加其他玩家授予访问权限 |
| **HUD 显示** | 看向被锁箱子时显示 "Locked by: 玩家名" |

### 保护机制

| 机制 | 说明 |
|------|------|
| **所有权验证** | 验证钥匙 UUID 与箱子记录的所有者是否匹配 |
| **密码保护** | 密码以哈希形式存储 |
| **服务端校验** | 所有锁定/解锁检查在服务端进行 |
| **防破坏** | 被锁箱子无法被破坏或移动 |
| **数据持久化** | 锁定信息自动保存，重启不丢失 |

### 使用方法

1. **创建钥匙**：手持 **绊线钩**，**Shift + 右键** 打开密码设置界面，输入密码生成钥匙
2. **锁定箱子**：手持钥匙，**右键** 点击未锁定的箱子
3. **解锁箱子**：手持钥匙，**Shift + 右键** 点击被自己锁定的箱子
4. **管理白名单**：
   - `/chestaccess add <玩家名>` — 添加白名单玩家
   - `/chestaccess remove <玩家名>` — 移除白名单玩家
   - `/chestaccess list` — 查看白名单

## 四、领地保护 — Banner Claim

Banner Claim（旗帜圈地）允许玩家通过放置旗帜和矿物块来声明和保护领地。

> **Modrinth**: [Banner Claim](https://modrinth.com/mod/banner-claim) | **MC百科**: [旗帜圈地](https://www.mcmod.cn/class/21930.html)

### 领地机制

#### 创建领地

将一个 **旗帜** 放置在一个 **完整的矿物块** 上即可创建领地。

#### 领地范围（以旗帜为中心的立方体）

| 底座矿物块 | 领地范围 |
|-----------|---------|
| **铜块** | 16 × 16 × 16 |
| **铁块** | 32 × 32 × 32 |
| **金块** | 64 × 64 × 64 |
| **钻石块** | 128 × 128 × 128 |
| **下界合金块** | 256 × 256 × 256 |

#### 领地可视化

- 按下 **`F3`** 查看已声明领地的边界
- 按下 **`H`** 键切换边界可视化显示

### 常用指令

| 指令 | 功能 |
|------|------|
| `/bc me` | 查看自己拥有的所有领地 |
| `/bc list` | 查看服务器所有领地（需OP） |
| `/bc rename <索引> <名称>` | 重命名领地 |
| `/bc unclaim <索引>` | 删除领地 |
| `/bc add <索引> <玩家>` | 添加领地成员 |
| `/bc remove <索引> <玩家>` | 移除领地成员 |

## 八、世界扩展 — Biomes O' Plenty

超多生物群系（Biomes O' Plenty）为主世界添加了大量全新的生物群系，大幅扩展了探索的多样性。

### 群系概览

BOP 新增了 50+ 种生物群系，分为以下几类：

| 类别 | 示例群系 |
|------|---------|
| **森林类** | 樱花林、枫树林、松林、竹林、薰衣草花海、魔法森林 |
| **湿地类** | 沼泽、红树林沼泽、泥滩 |
| **干旱类** | 沙漠、灌木地、旱地、草原 |
| **寒冷类** | 冻原、针叶林、雪山、冰川 |
| **山地类** | 高地、峰峦、石山、火山 |
| **热带类** | 热带雨林、珊瑚礁、海滩 |
| **水下类** | 海藻森林、珊瑚礁、深海 |
| **神秘类** | 魔法森林、神秘沼泽、高地荒地 |

### 新增方块与物品

| 类型 | 内容 |
|------|------|
| **木材** | 樱花木、枫木、松木、竹木、乌木、魔法木等 |
| **植物** | 薰衣草、金盏花、紫丁香、玫瑰丛、水葫芦等 |
| **石头** | 多种新石材，用于建筑装饰 |
| **染料** | 从新花朵中提取更多颜色 |

### 探索建议

1. BOP 世界广袤，善用 HuskHomes 的 `/rtp` 快速探索不同群系
2. 不同群系有独特的木材和建材，收集各色木材用于建筑
3. 部分群系有独特生物和资源，值得深入探索
4. 樱花林和魔法森林是热门的建筑选址
