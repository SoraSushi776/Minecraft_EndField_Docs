## 二、枪械系统

### 枪械类型

TaCZ 提供了涵盖多个时代的丰富枪械：

| 类型 | 示例 | 特点 |
|------|------|------|
| **手枪** | Glock、M1911、 desert eagle | 便携、单手、中近距离 |
| **冲锋枪 (SMG)** | MP5、UMP45、Vector | 高射速、机动性强 |
| **突击步枪 (AR)** | M4A1、AK47、SCAR | 全能型、适用各种距离 |
| **精确射手步枪 (DMR)** | SVD、MK14 | 半自动、中远距离 |
| **狙击步枪** | AWM、M200 | 高伤害、超远距离 |
| **轻机枪 (LMG)** | M249、RPK | 高弹容量、压制火力 |
| **霰弹枪** | SPAS-12、AA12 | 近距离高伤害、范围攻击 |
| **发射器** | RPG-7 | 爆炸性武器 |
| **近战武器** | 战术刀 | 快速近战 |

### CIBR 扩展枪包

CIBR 扩展包新增了大量现代化枪械，包括：

- **更多突击步枪**：HK416、G36C、FAMAS 等
- **PDW 系列**：P90、MP7 等
- **精确射击系列**：更多 DMR 和狙击步枪
- **定制配件**：CIBR 专属瞄准镜、握把等

## 三、配件系统

### 瞄准镜

| 类型 | 效果 |
|------|------|
| **红点/全息** | 近距离快速瞄准 |
| **2倍/4倍镜** | 中距离战斗 |
| **6倍/8倍/15倍镜** | 远距离狙击 |
| **夜视/热成像** | 低光环境作战 |

### 枪口配件

| 类型 | 效果 |
|------|------|
| **消音器** | 降低枪声，隐藏开火位置 |
| **制退器** | 减少后坐力 |
| **补偿器** | 提升精度 |
| **消焰器** | 消除枪口火焰 |

### 握把与枪托

| 类型 | 效果 |
|------|------|
| **垂直握把** | 减少垂直后坐力 |
| **直角握把** | 加快开镜速度 |
| **战术枪托** | 提升稳定性 |
| **轻型枪托** | 提升机动性 |

### 其他配件

- **战术导轨**：加装更多配件
- **战术手电**：照亮黑暗区域

- **扩容弹匣**：增加弹容量

## 五、枪械改装台

使用 **枪械改装台** 可以对枪械进行深度自定义：

1. 将枪械放入改装台
2. 选择要更换的配件槽位
3. 放入对应的配件物品
4. 完成改装，提升性能

---

## 六、战斗技巧

1. **距离把控**：不同枪械有最佳射程，霰弹枪近战、狙击枪远距离
2. **切换射击模式**：部分枪械支持单发/连发切换，灵活应对不同场景
3. **配件搭配**：根据战斗风格选择合适的配件组合
4. **弹药管理**：不同枪械使用不同弹药，注意携带足够的弹药
5. **消音器战术**：在探索时使用消音器避免吸引过多敌人
6. **快速换弹**：利用快速弹匣配件缩短换弹时间
