## 一、核心概念

### 动力（Rotation）

机械动力不使用电力或红石，而是通过 **旋转** 来驱动所有机器。转速用 **RPM**（每分钟转数）衡量。

### 应力（Stress）

每个机器工作时都会消耗应力（Stress），动力源提供应力容量。当总需求超过供给时，系统会 **超载（Overstressed）** 并停止工作。

### 动力网络

所有通过传动杆、齿轮、齿轮箱连接的机械共同构成一个动力网络，共享同一个转速和应力负载。

## 八、自动化流程设计

### 基础流水线构成

```
输入 ──→ 处理 ──→ 输出
 │          │        │
提取器    作业元件   传送带
 ↓          ↓        ↓
传送带 ──→ 动力源 ──→ 过滤器 → 板条箱
```

### 经典自动化方案

#### 1. 矿石粉碎线

```
水车 → 传动杆 → 粉碎轮(一对)
                    ↓ 输入：铁矿石
                    ↓ 输出：粉碎铁 + 副产物
                         ↓
                    传送带 → 板条箱
```

#### 2. 自动化农场

```
动力源 → 传动杆 → 动力收割机
                    ↓
             自动收割作物 → 传送带 → 板条箱
```

#### 3. 自动合成线

```
传送带(输入材料) → 动力制造器 → 传送带(输出成品)
                        ↓
                  需要动力接入
```

#### 4. 流体自动化

```
动力源 → 传动杆 → 动力泵 → 流体管道 → 工作盆
                               ↓
                         烈焰人燃烧室(加热)
                               ↓
                         动力搅拌器 → 成品
```

## 九、实用技巧

1. **转速匹配**：不同机器需要不同的 RPM，使用齿轮比调节（大齿轮带小齿轮加速，反之减速）
2. **应力管理**：先计算总应力需求，再确保动力源有足够容量
3. **模块化设计**：每个生产线独立建造，便于扩展和维护
4. **传送带转弯**：传送带可以转弯和上下坡，灵活设计物流路线
5. **动力分支**：使用齿轮箱从一个动力源分出多路动力
