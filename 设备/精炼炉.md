# 精炼炉

> 所有设备的合成需要借助 `制造台`，合成配方请参考原版游戏的耗材

## 介绍

使用高温精炼其他材料的设备。剥离、塑造……材料的形态将在高温中变换，直至焕然一新。

## 用途

- 高温精炼各类材料
- 将原材料加工为更高级的材料

### 参数

- **获取方式**：协议核心-设备制造
- **耗电功率值**：5

## 与原版MC的联动

精炼炉可替代高炉/熔炉处理大量原版物品：

**矿石冶炼**
- 各类矿石（铁、金、铜、锌、下界合金碎片、钻石、绿宝石、青金石、红石）→ 对应产物
- 粗矿（粗铁、粗金、粗铜）→ 对应锭

**工具/装备回收**
- 铁质/金质工具和盔甲 → 对应粒
- 马铠（铁/金）→ 对应粒

**材料加工**
- 沙子/红砂 → 玻璃
- 粘土球 → 红砖
- 圆石 → 石头
- 玫瑰石英 → 磨制玫瑰石英
