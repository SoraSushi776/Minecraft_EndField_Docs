# 种植机

## 介绍

自动化农业种植设备，能够高效种植各类作物。

## 用途

- 自动化种植各类农作物
- 提高农业产线效率

## 与原版MC的联动

种植机支持大量原版作物自动化种植：

**有种子作物**（种子 → 作物）
- 小麦、甜菜根、南瓜、西瓜、火把花、瓶子草

**无种子作物**（作物本身 → ×2）
- 胡萝卜、马铃薯、竹子、甘蔗、发光浆果、甜浆果、可可豆、下界疣、仙人掌

**花卉**（花 → ×2）
- 蒲公英、虞美人、兰花、绒球葱、蓝花美耳草、郁金香（4色）、雏菊、矢车菊、铃兰、凋零玫瑰

**树木**（树苗 → 原木）
- 橡树、云杉、白桦、丛林、金合欢、深色橡木、樱花、红树
