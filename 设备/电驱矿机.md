# 电驱矿机

![电驱矿机](https://patchwiki.biligame.com/images/zmd/b/bb/4bcm1j4h3m47qtw0mro4rpvxu7c0tnp.png)

> 所有设备的合成需要借助 `制造台`，合成配方请参考原版游戏的耗材

## 介绍

在通电状态下，可以用来开采源矿、紫晶矿的机械设备。因各项数据过于平均，该款设备在开拓区的销量并不好。

## 用途

- 开采源矿
- 开采紫晶矿

### 参数

- **获取方式**：协议核心-设备制造
- **耗电功率值**：5
- **协议容量需求**：2
- **占地需求**：3×3

## 与原版MC的联动

电驱矿机可开采多种原版石材：

**石材开采**（Tier 1）
- 石头 → 圆石
- 下界岩 → 下界岩
- 末地石 → 末地石
- 紫水晶母岩 → 紫水晶碎片
- 花岗岩 → 花岗岩
- 闪长岩 → 闪长岩
- 安山岩 → 安山岩
