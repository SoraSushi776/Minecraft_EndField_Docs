# 采种机

> 所有设备的合成需要借助 `制造台`，合成配方请参考原版游戏的耗材

## 介绍

用于采集作物种子的专用设备，保障农业生产的可持续发展。

## 用途

- 从成熟作物中采集种子
- 实现农业生产的自循环

## 与原版MC的联动

采种机可从原版作物和原木中获取种子：

**作物 → 种子**
- 小麦 → 小麦种子 ×2
- 甜菜根 → 甜菜种子 ×2
- 南瓜 → 南瓜种子 ×2
- 西瓜 → 西瓜种子 ×2
- 火把花 → 火把花种子 ×2
- 瓶子草 → 瓶子草荚果 ×2

**原木 → 树苗**
- 各类原木 → 对应树苗 ×2
- 红树木 → 红树繁殖体 ×2
