# 粉碎机

> 所有设备的合成需要借助 `制造台`，合成配方请参考原版游戏的耗材

## 介绍

能粉碎各种材料的重型设备。有没有听到它嚼碎硬物的声音？

## 用途

- 将各类材料粉碎为粉末形态
- 材料加工的前置步骤

### 参数

- **获取方式**：协议核心-设备制造
- **耗电功率值**：5

## 与原版MC的联动

粉碎机可处理大量原版方块和作物：

**石材粉碎**
- 圆石 → 沙砾 → 沙子
- 闪长岩 → 下界石英
- 花岗岩 → 红沙
- 安山岩 → 圆石
- 砂岩 → 沙子
- 陶瓦 → 红沙
- 滴水石块 → 粘土球
- 粘土块 → 粘土球 ×4

**植物加工**
- 羊毛 → 线 ×4
- 甜菜根 → 红色染料 ×2
- 仙人掌 → 绿色染料 ×2
- 可可豆 → 棕色染料 ×2
- 甘蔗 → 糖 ×2
- 小麦 → 小麦粉
- 各类花/蕨 → 对应染料
